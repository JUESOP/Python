<h3>문제</h3>
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.

<h3>입력</h3>
첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.

<h3>출력</h3>
N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

<h3>예제 입력 1</h3>
72
<h3>예제 출력 1</h3>
2
2
2
3
3

<h3>내가 작성한 코드</h3>
N = int(input())
i = 2

while N > 1:
    if N % i == 0:
        print(i)
        N //= i
        i = 2
    else:
        i += 1
